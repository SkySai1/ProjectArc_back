# Проектная структура

Данный файл описывает структуру проекта и назначение каждого файла или папки.

## Корневая структура

```
.
├── app
│   ├── routes              # Основные маршруты приложения
│   │   ├── about           # Маршруты для работы с описанием проекта
│   │   │   ├── create_about.py  # Создание описания проекта
│   │   │   ├── get_about.py     # Получение описания проекта
│   │   │   ├── __init__.py      # Инициализация маршрутов about
│   │   │   ├── README.md        # Документация маршрутов about
│   │   │   └── update_about.py  # Обновление описания проекта
│   │   ├── files           # Маршруты для работы с файлами
│   │   │   ├── create_file.py   # Создание файлов
│   │   │   ├── delete_file.py   # Удаление файлов
│   │   │   ├── __init__.py      # Инициализация маршрутов files
│   │   │   ├── read_file.py     # Чтение содержимого файлов
│   │   │   ├── README.md        # Документация маршрутов files
│   │   │   └── update_file.py   # Обновление файлов
│   │   ├── history         # Маршруты для работы с историей изменений
│   │   │   ├── get_history.py   # Получение истории изменений
│   │   │   ├── __init__.py      # Инициализация маршрутов history
│   │   │   ├── log_change.py    # Логирование изменений
│   │   │   ├── README.md        # Документация маршрутов history
│   │   ├── privacy         # Маршруты для политики конфиденциальности
│   │   │   ├── __init__.py      # Инициализация маршрутов privacy
│   │   │   ├── privacy_policy.py # Политика конфиденциальности
│   │   │   └── README.md        # Документация маршрутов privacy
│   │   ├── project_map     # Маршруты для работы с картой проекта
│   │   │   ├── delete_from_map.py  # Удаление записей из карты проекта
│   │   │   ├── get_project_map.py  # Получение карты проекта
│   │   │   ├── init_from_fs.py     # Синхронизация карты проекта с файловой системой
│   │   │   ├── __init__.py         # Инициализация маршрутов project_map
│   │   │   ├── README.md           # Документация маршрутов project_map
│   │   │   └── update_project_map.py # Обновление карты проекта
│   │   ├── __init__.py      # Инициализация маршрутов приложения
│   │   ├── models.py        # Модели данных для работы с базой данных
│   │   ├── README.md        # Документация для приложения
│   │   └── utils.py         # Утилиты для работы с проектом
├── tests                  # Тесты приложения
│   ├── conftest.py         # Фикстуры для тестов
│   ├── README.md           # Документация тестов
│   ├── test_about_auth.py  # Тесты аутентификации маршрутов about
│   ├── test_about_basic.py # Основные тесты маршрутов about
│   ├── test_about_update.py# Тесты обновления в маршрутах about
│   ├── test_files_auth.py  # Тесты аутентификации маршрутов files
│   ├── test_files_create.py# Тесты создания файлов
│   ├── test_files_delete.py# Тесты удаления файлов
│   ├── test_files_read.py  # Тесты чтения файлов
│   ├── test_files_update.py# Тесты обновления файлов
│   ├── test_history_auth.py# Тесты аутентификации маршрутов history
│   ├── test_history_file.py# Тесты файлов в истории изменений
│   ├── test_history_logging.py # Тесты логирования изменений
│   ├── test_privacy.py     # Тесты маршрутов privacy
│   ├── test_project_map_auth.py # Тесты аутентификации карты проекта
│   ├── test_project_map_basic.py# Основные тесты карты проекта
│   ├── test_project_map_errors.py# Тесты ошибок в маршрутах карты проекта
│   └── test_project_map_sync.py  # Тесты синхронизации карты проекта
├── config.py              # Конфигурация приложения
├── pytest.ini             # Конфигурация для Pytest
├── README.md              # Документация проекта
├── requirements.txt       # Список зависимостей
└── run.py                 # Главный файл для запуска приложения
```
